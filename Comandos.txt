Resumo dos comandos git - Miguel Alexandre da Cunha.

Acompanhar as modificações do repositório: 

$ git status 

Desfazer as modificações no arquivo README: 

$ git checkout -- README.md 

Adicionar as atualizações ao repositório: 

$ git add README.md 

Salvar todas as modificações realizadas: 

$ git commit -m "Teste de Apresentacao.md"

Apagar um arquivo anterior de teste: 

$ git rm Comandos.txt

Adicionar um novo  arquivo de texto: 

$ git add Comandos.txt

Garantir que a cópia esteja atualizada: 

$ git pull 

Atualizar as modificações locais: 

$ git push origin main

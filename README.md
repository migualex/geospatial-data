# Geospatial Data Science projects

This repository contains independent projects developed for applied remote sensing and machine learning tasks.

The codes included here cover:
- Flood mapping using multi-sensor satellite data (Sentinel-1 & Sentinel-2) in Google Earth Engine: `flood-mapping-gee/`
- Deep learning classification of açaí palm trees vs other palm species: `palm-tree-classification/`
- Cerrado deforestation monitoring methodology adaptation, including edge/border analysis: `cerrado-eu-regulation/`
